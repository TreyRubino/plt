# plt

A collection of projects in programming languages and compilers.

## Structure

**compilers/** - Compiler implementations including frontends, bytecode generation, and virtual machines.

**runtimes/** - Runtime systems including memory allocators, garbage collectors, and concurrency primitives.

**type_systems/** - Type system implementations exploring static enforcement of program properties.

**analysis/** - Program analysis tools including static analyzers, dataflow analysis, and formal verification.
